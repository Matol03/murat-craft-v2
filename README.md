# Murat Craft — портфолио (русская версия)

Минималистичный сайт-портфолио с акцентом на цвет: тёмная тема с фиолетовым и
вращающимся глобусом на фоне. Продажа шаблонов сайтов и приём индивидуальных
заказов. Чистый HTML/CSS/JS — без сборки, разворачивается где угодно.

## Запуск локально

Откройте `index.html` в браузере или поднимите локальный сервер:

```powershell
npm run dev
# затем откройте http://localhost:3000
```

## 1. Фотографии (нужны три)

Сайт использует ровно **три фотографии**. Превью сайтов в разделах «Работы» и
«Шаблоны» нарисованы на CSS, отдельные картинки для них не нужны.

| Файл | Где показывается |
|------|------------------|
| `assets/portrait.jpg` | Главный экран (вверху) |
| `assets/about.jpg`    | Раздел «Обо мне» |
| `assets/contact.jpg`  | Круглый аватар в разделе «Контакты» (а также иконка вкладки) |

## 2. Оплата через Stripe — уже подключена

Каждая кнопка «Купить» ведёт на **платёжную ссылку Stripe** и открывает оплату
в новой вкладке. Ссылки указаны прямо в `index.html` (ищите `buy.stripe.com`).

### ⚠️ Сейчас это ТЕСТОВЫЕ ссылки
В URL присутствует `test_` — это тестовый режим Stripe (реальные деньги не
списываются; для проверки используйте карту `4242 4242 4242 4242`, любые
будущие срок действия и CVC).

Перед приёмом реальных платежей:
1. Отключите **тестовый режим** в [панели Stripe](https://dashboard.stripe.com).
2. Создайте платёжные ссылки заново в «боевом» режиме.
3. Замените в `index.html` каждый `href` на «боевую» ссылку (без `test_`).

## 3. Кнопка «Связаться со мной»

Кнопка ведёт в Instagram, а под ней указан e-mail. Поменяйте ссылки в разделе
`#contact`, если нужно.

## Деплой (одной командой)

Проект настроен на деплой в **Vercel**:

```powershell
npm run deploy
```

При первом запуске выполните `vercel login` (откроется браузер), затем команда
будет публиковать обновления автоматически. Подробности — в оригинальном
репозитории.
